# Shelby Network Ecosystem (eco-hub)

This is a [Next.js](https://nextjs.org) project migrated from a high-fidelity static HTML landing page. It showcases the Shelby Network infrastructure, products, and real-time network metrics.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Global CSS with Cyberpunk/Tech aesthetic
- **Features**: Custom Cursor, Interactive Canvas, Scroll Reveals

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
